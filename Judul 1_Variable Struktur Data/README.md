Judul Program : Sistem Data Penjualan Buah

Program ini berfungsi sebagai sistem rekap atau manajemen simpel data penjualan buah yang dirancang untuk memudahkan pencatatan, pemantauan, dan evaluasi penjualan secara terstruktur. Algoritma dan struktur data yang diterapkan dalam program ini adalah List 2D atau matriks, di mana baris merepresentasikan setiap pembeli dan kolom merepresentasikan buahnya. Implementasi ini menggunakan teknik perulangan bersarang (nested loop) untuk mengakses dan mengolah data pada setiap sel matriks, yang kemudian dikombinasikan dengan mekanisme error handling untuk menjamin integritas input data serta teknik string formatting untuk menyajikan data dalam format tabel yang sistematis dan mudah dibaca.
